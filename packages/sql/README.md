# @mikro-orm/sql

Shared SQL driver base for [MikroORM](https://mikro-orm.io), built on top of Kysely. Provides the `QueryBuilder`, SQL connection management, and schema generation used by all SQL driver packages.

## Documentation

See the [official documentation](https://mikro-orm.io/docs/query-builder) and the [Kysely integration guide](https://mikro-orm.io/docs/kysely).
