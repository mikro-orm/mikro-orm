# @mikro-orm/core

TypeScript ORM for Node.js based on Data Mapper, Unit of Work and Identity Map patterns.

This is the core package of [MikroORM](https://mikro-orm.io), providing the `EntityManager`, metadata system, unit of work, identity map, and entity lifecycle management. Use it together with a [driver package](https://mikro-orm.io/docs/quick-start).
